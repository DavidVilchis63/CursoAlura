Nuevo proyecto 
        git init
        git init --bare (servidor remoto)
        git remote add servidor local "direccion"
        git remote -v 

Estatus 
        git status

Agregar  
        git add

Crear commit 
        git commit -m "Mensaje"

Desacer un cambio 
        git checkout --(Nombre de archivo)

Ver cambios 
        git diff (Nombre de archivo)

Ver commits 
        git log
        git log --oneline (resumido)
        git log -p (info extra) -> :q (para salir)

Ignorar archivos 
        Creando un archivo 
        .gitignore y escribir los nombres que 
        se quieren Ignorar (/ al final para carpetas)

Crear ramas 
        git branch (Nombre)

Ver ramas 
        git branch

Cambiar de rama 
        git checkout (Nombre de rama)

Subir repositorios a Github
        git remote add origin https://github.com/DavidVilchis63/PythonCurso.git
        git push -u origin main

Entrar en carpetas
        cd
        cd .. (salir una)
        mkdir Carpeta (crea carpeta)
